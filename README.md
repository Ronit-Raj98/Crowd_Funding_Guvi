# Crowd_Funding_Guvi
A robust, user-friendly crowdfunding platform developed using Java and supporting web technologies, designed to connect aspiring entrepreneurs, community projects, and changemakers with a network of enthusiastic supporters. This platform helps people launch campaigns, raise funds, and share their stories—from college startups and social initiatives to creative ventures—bridging the gap between ambition and opportunity.
A modern crowdfunding platform built with Java and web tech. Launch and manage fundraising campaigns, enable secure contributions, and support student projects, startups, and social causes through transparent, community-driven support.


### USAGE AND  FUNCTIONALITIES


##  Features
Secure Campaign Management: Easily create and manage fundraising campaigns, set funding goals, deadlines, and share campaign details with compelling descriptions and visuals.​
User Roles and Dashboards: Support for Admin, Campaign Creator, and Contributor roles, each with tailored dashboards for campaign monitoring, contribution tracking, and transparent reporting.
Seamless Contributions: Supporters can browse active campaigns, make secure donations, and track funding progress in real-time using integrated payment gateways (Razorpay/Stripe/PayPal) with a user-friendly interface.​
Real-Time Updates & Communication: Creators can post updates and connect with backers, fostering a sense of community and accountability throughout the campaign lifecycle.​
Advanced Security and Transparency: Built with robust Java backend and secure authentication (JWT, SSL encryption), ensuring trust for all users and transparent handling of funds.​
Full-Stack Java Implementation: The platform is built using Java for backend logic (Spring Boot/Servlets/JSP), SQL for data management, and HTML, CSS, JavaScript for a modern, responsive frontend

## User Functionalities
-**Register**: Users can create accounts to start or support campaigns.
-**Login**: Access and manage profiles securely.
-**Create Campaign**: Users can launch new crowdfunding campaigns with goals, deadlines, and descriptions.
-**Contribute to Campaigns**: Users can fund active campaigns by making secure donations.
-**View Contributions**: Users can track their donations and campaign support history.

## Campaign Creator Functionalities
-**Manage Campaigns**: Create, update, and close campaigns.
-**Track Funding Status**: View real-time progress toward funding goals.
-**Communicate with Backers**: Post updates and messages to campaign supporters.

## Admin Functionalities
-**User Management**: Add, edit, or delete user accounts (including campaign creators and contributors).
-**Campaign Oversight**: Monitor and manage all campaigns and contributions.
-**System Settings**: Configure roles, permissions, and platform settings.
-**Security**: Ensure data protection with encryption, authentication, and authorization.


## Technology Stack
-**Backend**: Java, Spring Boot, JDBC
-**Frontend**: HTML, CSS, JavaScript
-**Database**: MySQL

## Tools & Libraries:
-Spring Security (authentication and authorization)
-JDBC (database connectivity and operations)

## Prerequisites
-**Java Development Kit (JDK)**: Version 11 or later
-**Maven**: Dependency management
-**MySQL**: Database management system
-**IDE**: IntelliJ IDEA or Eclipse recommended
-**Git**: Version control system

Harnessing the power of collective support, this crowdfunding platform nurtures a culture of giving, collaboration, and empowerment. By combining secure technology with a user-focused experience, it inspires individuals and communities to innovate and drive meaningful change in society
